# Licenses

Основные правила этого resource pack:

- URL ресурспака не меняется.
- Новые narcotics assets добавляются только в namespace `assets/copimine/...`.
- Глобальные vanilla block textures не переопределяются.

## CopiMine-generated assets

В этом patch следующие assets являются self-made и принадлежат проекту CopiMine:

- narcotics item textures в `src/assets/copimine/textures/item/narcotics/`
- narcotics overlay textures в `src/assets/copimine/textures/gui/narcotics/`
- narcotics font manifest в `src/assets/copimine/font/narcotics_overlay.json`
- narcotics visual manifests в `src/assets/copimine/manifests/`
- narcotics shader/profile descriptor JSON в `src/assets/copimine/shaders/narcotics/`
- item model JSON и manifests для narcotics
- block visual marker textures/models, если не указано иное отдельным NOTICE

Источник:

- generated inside the CopiMine project
- generated by local project tooling and asset generation scripts

Статус лицензии:

- internal project assets for CopiMine
- no third-party proprietary asset embedded in this narcotics patch

## Optional client mod reuse

Клиентский мод `CopiMineClient` использует адаптированные копии уже существующих self-made overlay PNG из серверного resource pack:

- source: `resourcepacks/src/assets/copimine/textures/gui/narcotics/`
- target: `CopiMineClient/src/main/resources/assets/copimineclient/textures/visuals/`
- status: self-made internal CopiMine assets reused for optional client-side rendering

## Third-party review

Внешние permissive sources были просмотрены и задокументированы в:

- `THIRD_PARTY_NARCOTICS_VISUALS.md`

На текущем этапе сторонние visual assets в narcotics pack не встраивались.
