# CopiMine: что проверить перед запуском сервера

Этот документ нужен для быстрой проверки основных вещей. Здесь только самое важное.  
Разделы разделены на два блока:

1. Плагины
2. Сайт

---

## Плагины

### 1. AdminPlus

Проверь:

- Открывается главный админ-хаб.
- В хабе есть понятные разделы: выборы, экономика, игроки, миры и служебные разделы.
- Старые меню выборов больше не работают напрямую и уводят в новый модуль выборов.
- Команда жалобы и команда репорта не конфликтуют между собой.
- Безопасный скан AR открывается и не подвешивает сервер.

Основные команды:

- `/cmultra`
- `/cadm`
- `/ar`
- `/cmbank`
- `/appeal <текст>`
- `/report <текст>`
- `/rpguard`

Основные кнопки в GUI:

- Выборы
- Экономика
- Игроки
- Миры
- Проверки и служебные отчёты

### 2. EconomyCore

Проверь:

- Открывается раздел экономики.
- Банкоматы работают.
- Баланс игрока читается правильно.
- Переводы и выдача тестовых средств работают.
- Если банкомат не найден, не открывается лишнее меню и не сыпятся ошибки.

Основные функции:

- Банк игроков
- Личный счёт
- Казна президента
- ПИН-код
- Банкоматы
- История операций

### 3. ElectionCore

Проверь:

- Открывается новый раздел выборов.
- Создание участка работает через новый модуль.
- Старые выборные предметы не запускают старую логику.
- Этапы выборов меняются через новый сценарий, а не через старые кнопки.
- Участки, печати, бюллетени и результаты работают через новый модуль.

Основные функции:

- Создание выборов
- Участки ЦИК
- Печать ЦИК
- Заявки кандидатов
- Бюллетени
- Результаты
- Президент и казна

### 4. Artifacts

Проверь:

- Открывается лавка артефактов.
- Предметы за AR выдаются как официальные.
- Донатные предметы можно забрать в игре.
- Если предмет потерян внешне, его можно вернуть через отдельный экран.
- Если предмет сломан или израсходован по механике, бесплатный возврат не даётся.

Основные функции:

- Лавка за AR
- Донатная лавка в игре
- Получение купленных предметов
- Возврат утерянных предметов
- Проверка подлинности предмета

### 5. Narcotics

Проверь:

- Плагин загружается без ошибок.
- Официальные предметы работают только как положено.
- Готовые предметы нельзя запихнуть в запрещённые блоки обработки.
- Перегрузка и визуальные эффекты работают по текущим правилам.
- Команды администратора отвечают без ошибок.

Основные функции:

- Варка
- Официальные предметы
- Настройка визуальных эффектов
- Админ-команды

### 6. WorldCore

Проверь:

- Открывается раздел миров.
- Закрытие Nether и End просит подтверждение.
- Если список обычных миров пустой, всё равно выбирается рабочий основной мир.
- Перемещение в безопасную точку не кидает в лаву, пустоту или портал.

Основные функции:

- Ограничения миров
- Управление Nether
- Управление End
- Безопасное перемещение

## Сайт

### 1. Главные страницы

Проверь:

- Главная страница открывается отдельно.
- Страница сервера открывается отдельно.
- Страница лавок открывается отдельно.
- Страница модов открывается отдельно.
- Страницы не слеплены в один экран.
- На телефоне блоки не наезжают друг на друга.

### 2. Вход и регистрация

Проверь:

- Есть отдельная страница входа.
- Есть отдельная страница регистрации.
- Нет ручного выбора “игрок” или “администрация”.
- После входа сайт сам отправляет человека в нужный кабинет.

### 3. Личный кабинет игрока

Проверь:

- Открывается кабинет игрока.
- Видны баланс, история, покупки и нужные разделы.
- Можно открыть донатный раздел.
- Можно открыть раздел предметов.

### 4. Админка

Проверь:

- Открывается админский кабинет.
- Есть раздел регистрации админов.
- Есть разделы по балансу, покупкам, заявкам и выдаче.
- Младший админ не видит опасные действия, которые ему не положены.

### 5. Лавки на сайте

Проверь:

- Есть отдельная витрина за AR.
- Есть отдельная витрина за донатную валюту.
- Пополнение начинается на сайте.
- После оплаты покупка уходит в кабинет и дальше забирается в игре.
- Донат и AR не смешиваются.

### 6. Клиент и моды

Проверь:

- Страница модов открывается отдельно.
- Скачивается архив модов.
- В архиве лежат основные моды сервера.
- На странице есть версия, размер и контрольная сумма.

---

## Что проверять в первую очередь

Если времени мало, проверь сначала вот это:

1. Открывается админ-хаб.
2. Открывается новый раздел выборов.
3. Открывается экономика и банкомат.
4. Открывается лавка артефактов.
5. Работает вход на сайт.
6. Работает кабинет игрока.
7. Работает админка.
8. Работает скачивание архива модов.

---

## Что делать, если что-то сломано

- Если не открывается GUI, сначала проверь, загрузился ли нужный плагин.
- Если не открывается страница сайта, проверь адрес страницы отдельно.
- Если не выдаётся предмет, проверь инвентарь игрока и запись о покупке.
- Если не работает выборный сценарий, открывай только новый модуль выборов.
- Если есть сомнение, сначала не трогай опасные кнопки и проверь лог сервера.
